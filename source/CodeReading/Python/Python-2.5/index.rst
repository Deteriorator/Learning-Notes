此代码笔记大部分内容来自陈儒著的 《Python 源码剖析 —— 深度探索动态语言核心技术\
》 书籍。

此书中的 Python 版本是 Python-2.5， 代码压缩包可以从 Python 官网下载 \
`Python-2.5`_， 由于代码数量多， 比较大， 本仓库不附带 Python-2.5 代码， 有需要\
自行下载。

书中原文：

   当前 Python 的最新发布版本是 2.5.2，在本书中，我们剖析的对象是 2006 年 12 \
   月 19 日正式发布的 Python 2.5。
